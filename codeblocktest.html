Codeblock tests

The codeblock element poses many issues with output rendering. This topic tests many of these issues.

Codeblock inside notes

Inside a note, the shading of the note must be maintained but the shading of the codeblock is different. This poses a challenge. Each note type must be tested separately.

Note: This is a note with a code block inside a paragraph: 
First line of codeblock.
Second indented line of codeblock.
Third indented line of codeblock.
Tip: This is a tip with a code block (not in paragraph) in it: 
First line of codeblock.
Second indented line of codeblock.
Third indented line of codeblock.
Caution: This is a caution with a code block in it: 
First line of codeblock.
Second indented line of codeblock.
Third indented line of codeblock.

Codeblock containing varname

It does weird stuff when the varname tag is used as the first element within the codeblock or anyplace within the codeblock:

YourName is a wonderful thing.
Say it whenever you can.
It is an affirmation.  
Always repeat 
YourName when you are feeling lonely.
This is the last line in the code block.

This repeats the same codeblock, but puts it inside the introductory paragraph: 

YourName is a wonderful thing.
Say it whenever you can.
It is an affirmation.  
Always repeat 
YourName when you are feeling lonely.
This is the last line in the code block.

Codeblock with spaces for indentation

In some cases, spaces are used within a line to format code. The spaces are essential to the understanding, such as in XML markup: 

<CustomParameters>
    <SineParameters>
      <SineCyclePerDay>48</SineCyclePerDay>
      <SineAmplitude>100</SineAmplitude>
    </SineParameters>
    <RandomParameters>
        <RandomSpan>200</RandomSpan>
    </RandomParameters>
</CustomParameters>

Indentation can be critical in programming code: 

public class RandomCustomParameters : IPluginCustomParameters
{
  public SineParam SineParameters { get; set; }
  public RandomParam RandomParameters { get; set; }

  public class SineParam
  {
    public float SineCyclePerDay { get; set; }
    public float SineAmplitude { get; set; }
  }

  public class RandomParam
  {
    public float RandomSpan { get; set; }
  }
}
