Choice collection types (figures and tables)

Figures with descriptions

When you create a figure, an optional element that you include in the figure is the description element, . The description element describes the figure. At the most basic level, it might say "The following figure shows...." Text in the description element is always printed with the figure. The output script places the text immediately before the figure, separated from the title and figure box in help output.

The use of description elements is probably most useful if you reuse a figure in multiple topics. Then, each time you insert the figure, the description would automatically be included. You would not need to insert a separate paragraph or retype.

Figures without titles

Though titles are a good idea for figures (and necessary if you want to create a cross reference), you might have a figure without a title. In general, you probably would not have a description either, because such figures should flow from the text. But if you include a description, it will be output.
